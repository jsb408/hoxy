## 서비스 소개
HOXY는 사람이 필요한 사람들을 위해 모임을 개설하고 주변 사람들을 쉽게 모을 수 있도록 도와주는 Flutter 기반의 플랫폼 앱 입니다.
- 모임 모집 기능
- 모임원 간 채팅 기능
- 상호 간 리뷰 기능(예정)
